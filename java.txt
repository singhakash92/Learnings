Autoboxing is automatic conversion of primitives to wrapper objects, and unboxing is the reverse — both done by the compiler to bridge primitives and the object-oriented world of Java.
int num = 5;
Integer boxed = num;  // Autoboxing: int → Integer (compiler does Integer.valueOf(num))

Integer boxed = Integer.valueOf(10);
int num = boxed;  // Unboxing: Integer → int (compiler does boxed.intValue())

16) The this Keyword:
Sometimes a method will need to refer to the object that invoked it. To allow this, Java defines the this keyword.
this can be used inside any method to refer to the current object. That is, this is always a reference to the object on
which the method was invoked.

this -> is replaced by name of object if used along with variable or fun ex . this.zy or this.funname()
this -> name of class if used alone
